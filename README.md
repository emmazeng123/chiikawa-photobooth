# Chiikawa Photobooth

A cute polaroid-style photobooth featuring Chiikawa characters. Take two photos with your camera or upload your own, decorate with stickers, and save your polaroid.

## Features

- Camera mode with countdown timer and retake support
- Upload mode for using your own photos
- Sticker decorations — drag, resize, rotate, and delete
- Saves as a clean PNG with no handles or UI artifacts

## Running Locally

Camera access requires a local server — opening the HTML directly won't work.

```bash
npx serve
```

Then go to `http://localhost:3000` in your browser and allow camera access when prompted.
